"Automatic Backups are not supported"

If you are recieving this message while running a Google Doc in Advanced Services mode, this means you are likely using an outdated version of the service. Automatic Backups was added in version 0.3.1 beta. If you are running a version of Advanced Services past this point, your configuration may be set up to prevent automatic backups. Verify that the "allowBackups" variable is set to true. Otherwise, there may be a problem with the service or your backup target
